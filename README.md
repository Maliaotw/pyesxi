# pyesxi

## install

```
pip install pyesxi
```


## use

```
from pyesxi import dellemc_get_system_inventory
idrac = dellemc_get_system_inventory(
    idrac_ip=ip,
    idrac_port=port,
    idrac_user=idrac_user,
    idrac_passwd=idrac_passwd
)
```
